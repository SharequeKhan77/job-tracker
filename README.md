# Job Application Tracker

A web application built with Django to track job applications.

## Features
- Add, update and delete job applications
- Track application status (Applied, Interview, Rejected, Offer)
- Filter jobs by status
- Dashboard showing counts of each status
- Add notes to each application
- User authentication (login/logout)

## Tech Stack
- Python
- Django
- SQLite
- Bootstrap 5
- Git/GitHub

## How to Run
1. Clone the repository
2. Install dependencies: `pip install django`
3. Run migrations: `python manage.py migrate`
4. Create superuser: `python manage.py createsuperuser`
5. Run server: `python manage.py runserver`
6. Visit `http://127.0.0.1:8000`
